Rovio as a real time image processing source

The power of rovio is that it provides you an image continuously without any interruption and it transfers it quickly too. Since we have the real time image we can do a lot of image processing and computer vision stuff with rovio. Once you create the connection between your interface and MATLAB/OpenCV or any image processing tool then it will be a cake walk to implement the image processing algorithms. Here I have implemented edge detection algorithm and skin detection on the images streamed from rovio.
